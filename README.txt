Fetches pullrequests from bitbucket where the user is a reviewer and has not approved the pull request.

Requires NodeJS.

To use:

1. Clone the repo with: git clone https://github.com/cmodien/getMyOpenBitbucketPullRequests.git
2. run: npm install
3. run: node getMyOpenBitbucketPullRequests.js
4. Cmd + click on the urls output on the command line to open them in your browser.
